== Description ==

An add-on for MediaPress. Allow users to show MediaPress photos in light gallery view ( Full page slider ). User can select where he want to show media in light gallery view from MediaPress settings section.

== Installation ==

This section describes how to install the plugin and get it working.

You may need to follow the following steps.

1. Download `mpp-light-gallery.xyz.zip` (where xyz is version number )and extract
2. Upload `mpp-light-gallery` directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. That's all.
